# `paperclipai`

CLI package for onboarding, configuring, and running a Paperclip instance.

## Install

```bash
npm install -g paperclipai
```

Or run without a global install:

```bash
npx paperclipai onboard --yes
```

## Main Commands

- `paperclipai onboard`
- `paperclipai run`
- `paperclipai configure`
- `paperclipai doctor`
- `paperclipai plugin ...`
- `paperclipai worktree ...`
